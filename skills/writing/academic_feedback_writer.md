# Skill: Academic Feedback Writer

**Type:** Domain — Writing
**Applicable agents:** tfm_evaluator_agent, technical_writer_agent

## Purpose
Writes academic feedback that is direct, actionable, and defensible.

## Style
- Lead with the main verdict.
- Separate strengths from required changes.
- Explain why each change matters.
- Avoid vague encouragement and generic academic filler.
- Use respectful but explicit language.

## Structure
- **Verdict**
- **What works**
- **What must change**
- **Recommended rewrite or next action**
