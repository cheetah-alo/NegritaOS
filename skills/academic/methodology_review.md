# Skill: Methodology Review

**Type:** Domain — Academic
**Applicable agents:** tfm_evaluator_agent, proposal_validator_agent

## Purpose
Evaluates whether the proposed methodology can answer the stated objectives with available data and time.

## Checks
- Method type is appropriate for the research question.
- Data collection or dataset access is feasible.
- Evaluation criteria are defined before implementation.
- Risks, assumptions, and scope boundaries are explicit.
- The plan can be executed within the academic milestone.

## Output
Use: verdict, strengths, blocking gaps, recommended changes.
