# Skill: Dataset Validation

**Type:** Domain — Academic / ML
**Applicable agents:** tfm_evaluator_agent, proposal_validator_agent

## Purpose
Assesses whether a dataset is sufficient, ethical, and suitable for the proposed analysis.

## Checks
- Dataset owner, source, and access path are named.
- Target variable or outcome is defined when applicable.
- Required fields are known or auditable.
- Bias, privacy, and licensing risks are visible.
- Sample size and coverage are adequate for the claims.

## Rule
If the dataset is hypothetical or not yet accessible, mark conclusions as feasibility hypotheses.
